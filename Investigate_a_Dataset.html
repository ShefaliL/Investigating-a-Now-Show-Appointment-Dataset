Data Cleaning¶

Considering all the columns for data analysis , till date the analysed information tells us that:

1)There 14 columns in total out of which 13 are independent variables and 1 is dependent variable (No-show column). 2)The data set also does not contain any missing values(nan). 3)The column names have spelling mistakes which needs to be corrected. 4)The PatientId column should be integer value but it is in float.
